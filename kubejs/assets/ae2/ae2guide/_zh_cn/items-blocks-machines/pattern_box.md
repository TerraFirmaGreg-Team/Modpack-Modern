# 样板盒



样板盒与[网络工具](network_tool.md)类似，但用于储存与携带[已编码样板](patterns.md)，而非[升级卡](upgrade_cards.md)。

它共有 27 个槽位，用于存放[已编码样板](patterns.md)。只要样板盒位于物品栏任意位置，这些样板就会出现在任意 AE2 设备的界面中，可直接取用。
